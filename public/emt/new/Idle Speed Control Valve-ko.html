제품문의
Details
엔진 공회전 시 ECU의 제어 신호에 따라 공기 유량을 조절하여 안정적인 아이들 회전수를 유지하는 역할. 엔진이 공회전 상태일 때 흡입되는 공기량을 조절해 엔진 회전수를 일정하게 유지하는 장치로 운전자가 가속 페달을 밟지 않아도 엔진이 꺼지지 않고 안정적으로 작동 작동 방식 : 스테퍼 모터 구동형 밸브 작동 전압 : DC 12V 제어 신호 : 펄스 입력 밸브 스트로크 : 약 8 ~ 12mm 응답 시간 : ≤10ms 작동 온도 : -40 ℃ ~ +145 ℃ 커넥터 형식 : 4핀
