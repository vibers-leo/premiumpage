# Premiumpage 프로젝트 현황

## 기본 정보
- **프로젝트명:** Premiumpage (전자카탈로그 플랫폼)
- **프레임워크:** Next.js 16 + TypeScript
- **DB:** Prisma + SQLite (libsql)
- **인증:** NextAuth v5
- **배포:** Vercel (자동 배포)
- **도메인:** premiumpage.kr (서브도메인별 카탈로그)

## 현재 상태
- **단계:** 운영 중
- **주요 카탈로그:** HS-TECH, GENTOP, EMT, 항성산업사, 에어 HS-TECH
- **기능:** 전자카탈로그 뷰어, 관리자 대시보드, 견적 요청, PDF 변환
