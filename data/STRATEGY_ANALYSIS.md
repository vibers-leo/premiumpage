## 1. 현재 상태 진단

### 개발 완성도
- **코드 규모**: 3,253줄 (TypeScript) — 소규모 프로젝트
- **주요 기능**: 4개 산업용 카탈로그 (HS-TECH, GENTOP, EMT, 항성산업사) 완성, 멀티 언어 지원, 다크모드
- **기술 스택**: Next.js 16, Tailwind CSS v4, Framer Motion, Aceternity UI
- **상태**: 프로덕션 상태 (4개 카탈로그 배포 완료), 안정적 운영 중

### 매출/사용자 현황 (추정)
- **배포 상태**: 각 카탈로그별 도메인 라이브 (hs-tech.com, gentop.co.kr 등 추정)
- **사용자**: 각 카탈로그 월 500-2000 방문자 추정
- **매출**: 직접 매출 없음 (B2B 카탈로그용) — 고객사 매출 기여

### 핵심 과제
1. **수익화 모델 부재** — B2B 카탈로그는 매출이 아닌 마케팅 도구
2. **고객 확보 어려움** — 새로운 카탈로그 클라이언트 획득이 느림
3. **운영 복잡도** — 4개 카탈로그 × 다크모드 × 국영문 = 유지보수 부담
4. **기술 부채** — 에이전트 시스템(agent 스킬)이 복잡하고 관리 어려움

## 3. Beachhead 분석

**Primary Segment: 중견 산업용 제조사 (연매출 $10m-100m, 제품 50-500개)**

| 기준 | 평가 | 설명 |
|------|------|------|
| **Burning Pain** | ✅ 높음 | "PDF 카탈로그를 온라인으로 만들고 싶은데, 비용과 시간이 많이 든다" — 절실함 높음 |
| **Willingness to Pay** | ✅ 높음 | 카탈로그 개발 비용 ₩3-5m는 상대적으로 저렴. ROI: 해외 고객 3-5명 추가만 해도 회수 |
| **Winnability** | ✅ 높음 | 현재 4개 고객사 모두 성공적으로 운영 중. 신뢰도 높음 |
| **Referral Potential** | ⚠️ 낮음 | B2B 특성상 입소문이 약함. 하지만 "한국 제조사 동료"를 통한 소개는 가능 |

**2순위: 자동차/항공 부품 제조사**
- 더 높은 기술 요구사항 (CAD 연동, 공급사 관리)
- WTP 높음 (₩5m-10m)
- 하지만 정규화된 데이터 스키마 필요

## 4. Growth Loop

**B2B Sales-Driven Growth (PLG 불가능)**

```
영업팀 콜드 아웃리치
  ↓
"카탈로그 리뉴얼 데모" 보여주기
  ↓
프로젝트 의뢰 (₩2-5m)
  ↓
4주 개발 → 카탈로그 런칭
  ↓
고객사 경험 공유 (마케팅팀에서 링크 공유)
  ↓
경쟁사가 봤을 때 "우리도 저렇게 하고 싶다" → 새로운 기회
```

**핵심 메트릭**
1. 월간 신규 고객사 문의: 목표 3곳 이상
2. 프로젝트 성사율: 문의 → 계약 30% 이상
3. 고객사 만족도 (NPS): 70점 이상
4. 카탈로그 구글 SEO: 주요 키워드 1위 3개 이상

## 5. 가격 전략

### 현재 모델 (추정)
- **프로젝트 기반**: 카탈로그당 ₩2m-5m (1회)
- **유지보수**: 월 ₩500k (선택사항) — 계약률 낮음

### 권장 가격 조정

| 패키지 | 가격 | 포함 사항 |
|--------|------|----------|
| **Essential** | ₩2.5m | 영문만, 50개 상품까지, 기본 템플릿 |
| **Professional** | ₩4m | 국영문, 500개 상품, 커스텀 색상, 1개월 지원 |
| **Premium** | ₩6m | 국영문 + 3개국어, 무제한 상품, 다크모드 우선, 3개월 지원 |
| **Maintenance** | ₩300k/월 | 콘텐츠 업데이트, SEO 모니터링, 분석 리포트 |

### 수익화 전략 추가

1. **애드온 기능** (각 ₩500k-1m)
   - AI 기반 자동 제품 설명 생성
   - 다국어 자동 번역
   - CRM 연동 (리드 캡처)

2. **SaaS 구독 모델** (미래, 6-12개월 후)
   - 셀프서비스 카탈로그 빌더 (월 $199)
   - 템플릿 라이브러리
   - 24/7 지원

### 가격 검증 필요
- 현재 4개 고객사에 "유지보수" 제안 (기존 고객이므로 구매율 높음)
- 신규 고객 대상 "Professional" 기본 제시

## 7. North Star Metric

### 권장 NSM
**월간 고객 획득 비용 (CAC, Customer Acquisition Cost) 대비 생애 가치 (LTV) 비율**

- 정의: LTV / CAC 비율
- 목표: 12개월 내 LTV/CAC ≥ 3:1 달성
- 이유: (1) 사업 지속가능성 증명, (2) 스케일 가능 모델로의 전환 신호

### Input Metrics (월간 추적)
1. **신규 프로젝트 수주**: 월 1건 이상 (목표 3건)
2. **평균 프로젝트 가격**: ₩3.5m (목표)
3. **프로젝트 성사율**: 문의 → 계약 30% 이상
4. **고객 유지율**: 유지보수 구독 비율 30% 이상
5. **SEO 개선**: 주요 키워드 상위 3위 내 랭킹 5개 이상

## 8. 3단계 로드맵

### Phase 1: 기존 고객 유지보수 수익화 (현재~5월, 2개월)

**목표**: 4개 기존 고객에서 유지보수 구독 2곳 이상 확보, 월 ₩600k 추가 수익

**실행 항목**
- [ ] 기존 고객 4곳에 "유지보수 플랜" 제시 (월 ₩300k)
- [ ] 각 카탈로그 SEO 분석 + 최적화 (Google Search Console, 메타 태그 개선)
- [ ] 3개월 성과 분석 리포트 (방문자 수, 클릭률, 국가별 분석)
- [ ] 카탈로그별 사용자 피드백 수집 (NPS, 개선사항)

**성공 지표**
- 유지보수 구독 2곳 이상 확보 (월 ₩600k)
- SEO 개선으로 월 방문자 10% 증가
- NPS 70점 이상 유지

### Phase 2: 신규 고객 확보 & 가격 최적화 (5월~8월, 3개월)

**목표**: 신규 프로젝트 3건 이상, 월 ₩10m+ 수익

**실행 항목**
- [ ] 영업 아웃리치 (B2B 마케팅 에이전시 또는 파트타임 영업진)
- [ ] "카탈로그 사례 연구" 3개 작성 (HS-TECH, GENTOP 성과 공개)
- [ ] LinkedIn/업계 커뮤니티 아웃리치 (산업용 제조사 의사결정자)
- [ ] 프로젝트 개발 시간 최적화 (템플릿화로 4주 → 2주 단축)
- [ ] 신규 패키지 가격 테스트 (Essential ₩2.5m, Professional ₩4m)

**성공 지표**
- 신규 프로젝트 3건 이상 수주
- 월 누적 수익 ₩10m+ (프로젝트 + 유지보수)
- 프로젝트 평균 개발 기간 20% 단축

### Phase 3: SaaS 모델 전환 검토 (8월~2027년 1월, 6개월)

**목표**: SaaS 플랫폼 α 출시, 100명 이상 가입자

**실행 항목**
- [ ] "카탈로그 빌더" SaaS 프로토타입 개발 (Stripe 결제 통합)
- [ ] Freemium 모델 (무료: 템플릿, 유료: ₩99/월 Professional)
- [ ] 셀프서비스 온보딩 비디오 (20분)
- [ ] 커뮤니티 구축 (Slack/Discord)

**성공 지표**
- SaaS 가입자 100명 이상 (Free + Paid 포함)
- Paid 전환율 10% 이상
- 월 반복 수익(MRR) ₩10m+ (프로젝트 + SaaS 합산)

## 핵심 한 줄 요약

> **"제조사의 PDF 카탈로그를 SEO 최적화된 아름다운 웹 카탈로그로 변신시켜 글로벌 고객을 수익으로 전환"**

### 현재 위험도: 🟡 중간
- **강점**: 제품-시장 적합성 검증됨 (4개 고객 운영 중)
- **약점**: 프로젝트 기반이므로 스케일 한계, 영업 역량 부재
- **기회**: SaaS 전환으로 10배 성장 가능

### 투자 결정 기준
1. **양수 신호**: Phase 1에서 유지보수 2곳 확보 + Phase 2에서 신규 3건 수주 → SaaS 투자 검토
2. **음수 신호**: 신규 고객 문의 월 1건 미만 지속 → 마케팅 재전략 필요
3. **재검토 신호**: SaaS 전환 검토 시 경쟁사(Issuu, Canva) 분석 필수

---

**최종 조언**:
이 프로젝트는 **가장 안정적인 프로젝트**입니다. 이미 4개 고객이 있고, 제품도 완성됐으며, 고객도 만족하고 있습니다. 하지만 **장기 성장**을 위해서는:

1. **단기 (3개월)**: 기존 고객 유지보수 구독화로 월 ₩500k-1m 추가 수익 창출
2. **중기 (6개월)**: 영업을 통해 신규 프로젝트 수주로 월 ₩15m 이상 수익 달성
3. **장기 (12개월)**: SaaS로 전환하여 스케일 제한 극복

지금 바로 할 일은 **기존 고객 4곳에 유지보수 제안** (회신율 높음) + **신규 프로젝트 파이프라인 구축** (영업팀 충원)입니다.
